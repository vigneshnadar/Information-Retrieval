     <p>
      <b>
       Metre per second
      </b>
      (
      <a href="/wiki/American_English" title="American English">
       American English
      </a>
      :
      <b>
       meter per second
      </b>
      ) is an
      <a href="/wiki/SI_derived_unit" title="SI derived unit">
       SI derived unit
      </a>
      of both
      <a href="/wiki/Speed" title="Speed">
       speed
      </a>
      (
      <a href="/wiki/Scalar_(physics)" title="Scalar (physics)">
       scalar
      </a>
      ) and
      <a href="/wiki/Velocity" title="Velocity">
       velocity
      </a>
      (
      <a class="mw-redirect" href="/wiki/Vector_(geometry)" title="Vector (geometry)">
       vector quantity
      </a>
      which specifies both magnitude and a specific direction), defined by
      <a href="/wiki/Distance" title="Distance">
       distance
      </a>
      in
      <a href="/wiki/Metre" title="Metre">
       metres
      </a>
      divided by
      <a href="/wiki/Time" title="Time">
       time
      </a>
      in
      <a href="/wiki/Second" title="Second">
       seconds
      </a>
      .
     </p>
     <p>
      The
      <a href="/wiki/International_System_of_Units" title="International System of Units">
       SI unit
      </a>
      symbols are m·s
      <sup>
       −1
      </sup>
      , m s
      <sup>
       −1
      </sup>
      , m/s, or
      <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;">
       <span style="display:block; line-height:1em; margin:0 0.1em;">
        m
       </span>
       <span class="visualhide">
        /
       </span>
       <span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">
        s
       </span>
      </span>
      ,
      <sup class="reference" id="cite_ref-1">
       <a href="#cite_note-1">
        [1]
       </a>
      </sup>
      sometimes (unofficially) abbreviated as "mps". Where metres per second are several orders of magnitude too slow to be convenient, such as in
      <a href="/wiki/Astronomy" title="Astronomy">
       astronomical
      </a>
      measurements, velocities may be given in kilometres per second, where 1 km/s is 1000 metres per second, sometimes unofficially abbreviated as "kps".
     </p>

     <h2>
      <span class="mw-headline" id="Conversions">
       Conversions
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Metre_per_second&amp;action=edit&amp;section=1" title="Edit section: Conversions">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      1 m/s is equivalent to:
     </p>
     <dl>
      <dd>
       = 3.6
       <a href="/wiki/Kilometres_per_hour" title="Kilometres per hour">
        km/h
       </a>
       (exactly)
      </dd>
      <dd>
       ≈ 3.2808
       <a class="mw-redirect" href="/wiki/Feet_per_second" title="Feet per second">
        feet per second
       </a>
       (approximately)
      </dd>
      <dd>
       ≈ 2.2369
       <a href="/wiki/Miles_per_hour" title="Miles per hour">
        miles per hour
       </a>
       (approximately)
      </dd>
      <dd>
       ≈ 1.9438
       <a href="/wiki/Knot_(unit)" title="Knot (unit)">
        knots
       </a>
       (approximately)
      </dd>
     </dl>
     <p>
      1
      <a class="mw-redirect" href="/wiki/Feet_per_second" title="Feet per second">
       foot per second
      </a>
      = 0.3048 m/s (exactly)
     </p>
     <p>
      1
      <a href="/wiki/Miles_per_hour" title="Miles per hour">
       mile per hour
      </a>
      = 0.44704 m/s (exactly)
     </p>
     <p>
      1
      <a href="/wiki/Kilometres_per_hour" title="Kilometres per hour">
       km/h
      </a>
      = 0.2
      <span style="text-decoration:overline;">
       7
      </span>
      m/s (exactly)
     </p>
     <p>
      1 kilometre per second is equivalent to:
     </p>
     <dl>
      <dd>
       ≈ 0.6213 miles per second (approximately)
      </dd>
      <dd>
       ≈ 2237 miles per hour (approximately)
      </dd>
     </dl>
     <h2>
      <span class="mw-headline" id="Relation_to_other_measures">
       Relation to other measures
      </span>
      <span class="mw-editsection">
       <span class="mw-editsection-bracket">
        [
       </span>
       <a href="/w/index.php?title=Metre_per_second&amp;action=edit&amp;section=2" title="Edit section: Relation to other measures">
        edit
       </a>
       <span class="mw-editsection-bracket">
        ]
       </span>
      </span>
     </h2>
     <p>
      The benz, named in honour of
      <a href="/wiki/Karl_Benz" title="Karl Benz">
       Karl Benz
      </a>
      , has been proposed as a name for one metre per second.
      <sup class="reference" id="cite_ref-Klein2011_2-0">
       <a href="#cite_note-Klein2011-2">
        [2]
       </a>
      </sup>
      Although it has seen some support as a practical unit,
      <sup class="reference" id="cite_ref-Heijungs2005_3-0">
       <a href="#cite_note-Heijungs2005-3">
        [3]
       </a>
      </sup>
      primarily from German sources,
      <sup class="reference" id="cite_ref-Klein2011_2-1">
       <a href="#cite_note-Klein2011-2">
        [2]
       </a>
      </sup>
      it was rejected as the SI unit of velocity
      <sup class="reference" id="cite_ref-Cardarelli2004_4-0">
       <a href="#cite_note-Cardarelli2004-4">
        [4]
       </a>
      </sup>
      and has not seen widespread use or acceptance.
      <sup class="reference" id="cite_ref-Dresner1974_5-0">
       <a href="#cite_note-Dresner1974-5">
        [5]
       </a>
      </sup>
     </p>
